## Vibe Coding 开发理念

**什么是 Vibe Coding？**

Vibe Coding 是一种**以灵感和创造力为核心的开发方式**：

1. **灵感驱动 / Inspiration-Driven**
   - 让创意自由流动
   - 不被过度规范束缚
   - 快速迭代，拥抱变化

2. **用户中心 / User-Centric**
   - 从用户真实需求出发
   - 体验优先于架构
   - 快速验证假设

3. **敏捷迭代 / Agile Iteration**
   - 小步快跑
   - 每日交付价值
   - 持续学习优化
